# Source Code Praktikum — Pengembangan Aplikasi Web (TI-501)

Proyek ini tumbuh bertahap mengikuti urutan RPS. Setiap pertemuan = satu commit git,
sehingga urutan pengembangan bisa ditelusuri lewat riwayat commit:

```bash
git log --oneline --reverse
```

Lihat `/panduan` (folder terpisah, sudah dibagikan sebelumnya) untuk instruksi langkah-demi-langkah tiap pertemuan. Folder ini berisi **hasil akhir kode itu sendiri**, bukan penjelasannya.

- `backend/` — Node.js + Express + Prisma + JWT (Pertemuan 2–5, 10, 11, 13, 14, 15)
- `frontend/` — ReactJS + Vite (Pertemuan 7, 9, 12, 14)
